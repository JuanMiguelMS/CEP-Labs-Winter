# CEP-Labs-Winter

## Project Overview
A TTC-style transit departure board written in C. Reads route data from a file and displays a live countdown terminal UI using ANSI escape codes.

## Files
- `departure_board.c` — Main C source file with structs, functions, and countdown logic
- `routes.txt` — Route data: `<route_num> <name> <minutes_until_arrival>`
- `departure_board` — Compiled binary (generated by gcc)

## How It Works
1. Loads routes from `routes.txt` (route number, name, initial arrival time in minutes)
2. Converts minutes to a per-second countdown
3. Redraws the board every second using ANSI escape codes (`\033[2J\033[H`)
4. Red TTC-style color theme; routes arriving in ≤ 1 minute turn yellow with `<-- NOW`
5. Departed routes (countdown hits 0) are shown dimmed as `** DEPARTED **`

## Running
```bash
gcc -o departure_board departure_board.c && ./departure_board
```

## routes.txt Format
One route per line:
```
<route_number> <route_name> <minutes_until_arrival>
```
Example:
```
501 Queen 5
504 King 12
```
